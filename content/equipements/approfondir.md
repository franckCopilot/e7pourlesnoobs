sections:
  - sectionTitle: "Présentation"
    content: |
      Les équipements sont un élément indispensable sur Epic Seven pour améliorer la puissance de vos héros. Dans ce tutoriel, nous allons découvrir les bases de la fabrication d'équipements, les rolls, les différents archétypes de héros ainsi que les combinaisons de caractéristiques à rechercher pour un équipement.
  - sectionTitle: "Fabrication d'Équipement"
    content: |
      La fabrication d'équipement est quelque chose que vous allez effectuer très régulièrement. Il y a deux principales sources de fabrication :

      - **La fabrication standard** — Se trouve dans Sanctuaire → Atelier d'acier. Il faudra utiliser des gold + des matériaux de fabrication.
      - **La fabrication améliorée** — Se trouve dans Sanctuaire → Clocher de l'alchimiste. Requiert des gemmes de fabrication + de la matière d'origine.

      La règle basique : favorisez la **fabrication standard** pour les équipements de **gauche** (Arme, Casque, Armure) et la **fabrication améliorée** pour le **côté droit** (Collier, Anneau, Bottes).

      Les équipements de gauche ont une caractéristique principale fixe. Les équipements du côté droit peuvent avoir une caractéristique principale **aléatoire**.

      La fabrication améliorée vous permet de choisir la caractéristique principale et le set de votre équipement.

  - sectionTitle: "Amélioration / Roll d'Équipement"
    content: |
      **Rolls possibles pour un équipement niveau 85 Épique (rouge) :**

      - Santé : 157-202 / 4%-8%
      - Attaque : 33-46 / 4%-8%
      - Défense : 28-35 / 4%-8%
      - Vitesse : 2-5* (*5 a seulement 0,3% de chance)
      - Chance de coup critique : 3%-5%
      - Dégâts en coup critique : 4%-7%
      - Résistance d'effet : 4%-8%
      - Efficacité : 4%-8%

      Les **points d'équipement (PE)** sont une valeur numérique basée sur la somme des caractéristiques d'une pièce d'équipement. La règle générale est de continuer d'améliorer un équipement en +9 si vous avez **55/60+ de PE**.

      En général, il est recommandé d'éviter les caractéristiques flat (sans pourcentage). Dans de rares cas, la défense flat est meilleure que la défense % (souvent sur les assassins).

      Idéalement, vous devriez roll seulement les équipements qui ont un Set + une caractéristique principale + 3 caractéristiques secondaires qui synergisent. 4 caractéristiques secondaires qui synergisent serait parfait, mais 3 est déjà acceptable.

      Il est recommandé d'estimer le potentiel d'un équipement une fois en +9 pour vérifier s'il a bien roll dans les caractéristiques souhaitées.

      **Objectifs généraux de roll (supérieur à la moyenne) :**

      - **6%+** pour l'attaque, la santé, la défense, l'efficacité, l'ER
      - **5%+** pour les dégâts en coup critique
      - **3+** pour la vitesse
      - **4%+** pour les chances de coup critique
  - sectionTitle: "DPS"
    content: |
      Les héros DPS sont destinés à infliger des dégâts. Ils sont généralement vulnérables mais ont besoin d'attaque, de vitesse, de chances de coup critique et de dégâts en coup critique.

      **Caractéristiques à rechercher pour les DPS :**

      - Arme : Attaque %, Vitesse, Chances Crit, Dégât Crit, Santé %
      - Casque : Attaque %, Vitesse, Chances Crit, Dégât Crit, Santé % ou Défense %
      - Armure : Vitesse, Chances Crit, Dégât Crit, Santé % ou Défense %
      - Collier (**principal** : Chances Crit ou Dégât Crit) : Attaque %, Vitesse
      - Anneau (**principal** : Attaque %) : Chances Crit, Dégât Crit, Vitesse
      - Bottes (**principal** : Vitesse ou Attaque %) : Chances Crit, Dégât Crit
  - sectionTitle: "Bruiser"
    content: |
      Les héros bruiser sont destinés à infliger des dégâts tout en ayant des caractéristiques défensives leur permettant de survivre plus longtemps. La plupart des caractéristiques sont bien sur eux et ils peuvent utiliser la plupart des équipements.

      **Caractéristiques à rechercher pour les Bruiser :**

      - Arme : Attaque %, Vitesse, Chances Crit, Dégât Crit, Santé %
      - Casque : Attaque %, Vitesse, Chances Crit, Dégât Crit, Santé %, Défense %
      - Armure : Vitesse, Chances Crit, Dégât Crit, Santé %, Défense %
      - Collier (**principal** : Chances Crit, Dégât Crit, Santé % ou Défense %) : Attaque %, Vitesse
      - Anneau (**principal** : Attaque %, Santé % ou Défense %) : Chances Crit, Dégât Crit, Vitesse
      - Bottes (**principal** : Vitesse, Attaque %, Santé % ou Défense %) : Chances Crit, Dégât Crit
  - sectionTitle: "Tank / Healer"
    content: |
      Les héros tank et healer ont des équipements similaires. Ces héros veulent basiquement survivre le plus longtemps possible et proposer de la survie au reste de votre équipe.

      **Caractéristiques à rechercher pour les Tank / Healer :**

      - Arme : Vitesse, Santé %, Résistance d'effet, Efficacité
      - Casque : Vitesse, Santé %, Défense %, Résistance d'effet, Efficacité
      - Armure : Vitesse, Santé %, Défense %, Résistance d'effet, Efficacité
      - Collier (**principal** : Santé % ou Défense %) : Vitesse, Résistance d'effet, Efficacité
      - Anneau (**principal** : Santé %, Défense % ou Résistance d'effet) : Vitesse, Efficacité
      - Bottes (**principal** : Vitesse, Santé % ou Défense %) : Résistance d'effet, Efficacité
  - sectionTitle: "Buffer / Debuffer"
    content: |
      Les héros buffer et debuffer ont pour rôle d'appliquer des bonus à leurs alliés ou des malus à leurs adversaires. La vitesse est un facteur important pour jouer plus souvent.

      **Caractéristiques à rechercher pour les Buffer / Debuffer :**

      - Priorité à la **vitesse** pour jouer fréquemment
      - **Efficacité** élevée pour maximiser les chances d'infliger des malus
      - Santé %, Défense % pour la survie
      - Bottes vitesse obligatoires dans la plupart des cas
